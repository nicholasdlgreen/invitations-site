SEO Rules Reference

Key rules to check after each update. Use as a checklist when auditing the site.

On-Page Fundamentals
  • ✦ One H1 per page, contains primary keyword
  • ✦ Title tag: 50–60 chars, keyword near start
  • ✦ Meta description: 140–160 chars, CTA included
  • ✦ URLs: short, lowercase, hyphenated, keyword-rich
  • ✦ Image alt text on every image
  • ✦ Internal links between related pages
  • ✦ No duplicate title tags or meta descriptions
E-E-A-T Signals (Dec 2025+)
  • ✦ Named author/brand on all content pages
  • ✦ Real testimonials with names and dates
  • ✦ About page explaining who you are
  • ✦ Contact information visible
  • ✦ Content demonstrates genuine expertise
  • ✦ No anonymous or generic AI-only content
  • ✦ Regular content freshness — update dates
Technical SEO
  • ✦ Each page has unique URL (no SPA)
  • ✦ sitemap.xml submitted to Search Console
  • ✦ robots.txt allows Googlebot
  • ✦ HTTPS on all pages
  • ✦ LCP < 2.5s, CLS < 0.1
  • ✦ Mobile-first responsive design
  • ✦ JSON-LD structured data (Organisation + Product)
PPC Quality Score
  • ✦ Ad keywords must match landing page content
  • ✦ CTR history improves score over time
  • ✦ Quality Score 10 = 50% CPC discount
  • ✦ Use Responsive Search Ads (only format now)
  • ✦ Performance Max for scaling (needs 50+ conversions)
  • ✦ Enhanced CPC deprecated March 2025
  • ✦ Consent Mode v2 required for UK/EU campaigns
